#   MERN BLOGGY APP  -  WEEK SEVEN PROJECT !!

dis is very big project we do in week 7 .  it have  backend  and  frontend  side  all work  together .
we use  mern stack  which mean  mongodb  and  express  and  react  and  node js  magic .

---

##   Wat we do  in dis  folder :

-   we mak  REST api  with express js  so we can  talk  to database  and  frontend . 
-   react  is use for  mak  the face  of the app  so people can  see  the blog . 
-   we do  login  and  signup  with  secret cookies  name  jwt  token . 
-   author people can  write  new stories  and  edit  them  if they  mak  mistake . 
-   all stories  and  user info  go  to  mongodb  cloud  so it  never  lost . 
-   we use  cloudinary  and  multer  for  upload  the selfies  and  blog pics  to  the internet cloud . 

---

##  Foldr Structure  (Dont get confuse) :

-   Backend-BlogApp :   dis  is  the brain  where  server  run  and  db  connect . 
    -   APIs :   many  portals  for  admin  and  author  and  user  peeps . 
    -   models :   skeletons  for  how  data  look  in  mongo . 
    -   server.js :   the main  mainframe  that  start  everything  on  port 5000 . 
-   Frontend-BlogApp :   dis  is  the pretty  side  that  user  see  on  browser . 
    -   components :   many  small  lego  blocks  like  login  and  write  story . 
    -   stores :   where  react  remember  who  is  login  right  now . 

---

##   Key Learnings  (Wat we learn  today) :

-   mern is  very  powerful  but  sometime  it  give  error  name  cors  or  network . 
-   bouncers  are  good  for  check  if  user  is  allow  to  see  the secret  page . 
-   cloudinary  is  like  magic  bucket  for  images  on  the web . 
-   always  end  with  final  submitt  so  teacher  is  happy .

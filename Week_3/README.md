#   ATP  -  Week  3  (Backend  and  Database) !!

dis  foldr  is  where  we  start  building  the  back  side  of  the  application  using  node  and  mongo . 
we do  practice  for  mongodb  queries  and  mak  a  user  server . 

---

##   What  we  learn :

-   we learn  mongodb  fundamentals  and  how  to  store  data  in  collections  and  documents . 
-   we do  practice  for  CRUD  operations  (create  read  update  delete)  using  mongo  shell . 
-   express  js  server  is  mak  to  handle  the  http  requests  like  GET  and  POST . 
-   we connect  the  backend  server  to  the  mongo  cloud  using  the  connection  string . 

---

##   Files  in  dis  Folder :

-   MongoDbPractice.txt :   all  the  queries  we  run  in  the  database  shell . 
-   server.js :   the  backend  script  that  listens  for  the  users  coming  in . 
-   models :   where  we  define  how  the  user  and  product  data  should  look . 
-   reqq.http :   a  file  to  test  if  the  server  is  yelling  back  correctly .

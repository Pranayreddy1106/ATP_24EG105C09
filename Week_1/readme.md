#   ATP  -  Week  1  (The  Start  of  Journey) !!

dis foldr  is  where  everything  start  for  learning  javascript  and  web  things . 
we do  many  topics  on  day 1  and  day 2  and  solve  many  problems . 

---

##   Day 1  -  Web  and  JS  Basics :

-   we learn  how  the internet  works  with  client  and  server  talking  to  each other . 
-   we learn  javascript  data  types  like  numbers  and  strings  and  booleans . 
-   we do  control  flow  with  if  else  and  loops  like  for  and  while . 
-   many  functions  we learn  like  arrow  functions  and  expressions . 

###   Day 1  HW :
1.   find  which  num  is  bigger  between  two  input . 
2.   find  the biggest  of  three  num  (math is hard) . 
3.   sum  all  the marks  in  a  list  of  numbers . 
4.   find  the tiny  smallest  element  in  marks  list . 

---

##   Day 2  -  Advanced  JS  Pro  Stuff :

-   we deep  dive  into  functions  and  something  name  closures  (keep secrets) . 
-   we learn  callback  functions  so  one  func  can  call  another  func . 
-   objects  and  arrays  are  very  important  so  we learn  how  to  update  them  dynamically . 
-   we work  with  list  of  employees  and  change  their  data . 

###   Day 2  HW :
1.   put  new  employee  at  the  second  place  in  the list . 
2.   remove  the guy  name  "Kiran"  because  we  dont  want  him . 
3.   change  the marks  of  Sneha  from  95  to  75  because  she  was  sleeping . 

---

dis is  very  good  start  for  us  to  become  pro  developer .

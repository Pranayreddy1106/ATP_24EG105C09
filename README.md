##   IMP POINTS (Very important) :

-   **MERN Stack Magic** : we use mongodb and express and react and node js to make dis app very fast and strong . 
-   **Secret Security** : we use JWT tokens and cookies for login so bouncers can check if user is real or fake . 
-   **Cloud Storage** : all images are going to cloudinary bucket so we dont waste server space . 
-   **Responsive Design** : app look very pretty on phone and tablets also because we use tailwind css . 
-   **Clean Code** : folder are separated by weeks so anyone can see how we grow from week 1 to week 7 . 

---

##   WORKFLOW OF THE APP (How it works) :

1.  **Home Page** : when user open app they see many beautiful blogs from different authors . 
2.  **User Login** : if you want to write a story you must login . if you dont have account you can signup first . 
3.  **Author Dashboard** : after login you go to secret dashboard where you can see all your stories . 
4.  **Write Blog** : click on write button to make new blog . you can add title and big content and also upload one nice photo . 
5.  **Save to Cloud** : when you click submit the photo goes to cloudinary and text goes to mongodb atlas cloud . 
6.  **Edit / Delete** : if you make typo mistake you can edit your blog or delete it if you dont like it anymore . 
7.  **Logout** : after work is finish user can logout and the jwt token is clear from browser .

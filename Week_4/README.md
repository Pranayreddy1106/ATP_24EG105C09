#   ATP  -  Week  4  (HTML  and  CSS  Design) !!

dis  foldr  is  where  we  learn  how  to  make  the  skeleton  and  skin  of  the  website . 
we use  html  for  structure  and  css  for  making  it  look  not  so  ugly . 

---

##   Topics  We  Learn :

-   we learn  all  the  basic  tags  like  div  and  p  and  h1  for  writing  text . 
-   nested  elements  are  important  so  we  learn  how  to  put  box  inside  box . 
-   we build  a  very  cute  cat  page  to  practice  images  and  links . 
-   css  styles  are  learn  to  change  colors  and  fonts  and  spacing . 
-   we do  forms  so  user  can  type  their  names  and  secret  info . 

---

##   Files  in  dis  Folder :

-   HtmlElements.html :   a  simple  page  with  many  tags  we  learn . 
-   NestedHtmlElements.html :   practice  for  putting  things  inside  other  things . 
-   Cat.html :   the  best  page  about  cats  on  the  web . 
-   style.css :   where  we  keep  all  the  painting  instructions  for  html .
